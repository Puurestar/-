**抵达～**

> 记得要采一叶一芽哦 越嫩越好！

> 采来的茶叶不能碰水哦，不然就报废啦

``` text
采茶时 需要用手 轻轻地 揪茶叶
不要用指甲掐哦
否则指甲里的汗液会氧化茶叶中的有效成分 导致质量下降哦
——————
成品茶需要用70-80摄氏度的水泡一遍，需要倒掉
不能用刚煮沸的开水哦
否则也会导致口味受到损失
```
